参考示例图片，绘制一张清晰的游戏角色精灵图（spritesheet）。

角色身份/定位：{{characterRole}}
角色外观重点：{{characterAppearance}}
世界视觉背景（仅供参考）：{{worldVisualContext}}
所属 IP / 作品：{{ipSource}}

采用Q版风格。精灵图以严格的网格布局排列在纯绿色背景上。图中没有文字、网格线或数字。

具体内容：
- 第1行：向左走（6帧），从左到右依次排列，每帧人物姿态连续变化形成行走动画
- 第2行：向下走/正面朝向屏幕走（6帧），同上
- 第3行：向上走/背对屏幕走（6帧），同上
- 第4行：三个不同方向下的静态站立图，第一列为正对屏幕站立，第二列为背对屏幕站立，第三列为向左侧对屏幕站立，第三列往后空白
- 第5行：完全空白

注意：
- 你生成的图片中的每一格中的人物动作必须与参考图中的对应动作一致。
- 风格为简洁的像素艺术，宽高比为1:1。
- 背景必须为纯绿色（#00B000 附近），不要有其它颜色的背景杂质。
- 每帧人物大小一致，位置居中在所属网格内。
- 参考图只是告诉你整体布局结构、人物动作（需严格遵循），但人物的具体长相、表情、着装、身高、体型等需按照角色身份与外观重点绘制。
- 世界视觉背景只用于帮助你理解整体时代/环境气质，不是硬约束。
- 如果角色设定本身与世界背景存在反差或错位（例如古代世界中的现代穿越角色、未来世界中的复古角色），必须优先保留角色自身设定，不要为了迎合世界背景而抹平这种反差。
- 角色自身设定的优先级高于世界视觉背景：先看"角色身份/定位"和"角色外观重点"，再将世界背景作为辅助参考。
- **IP 角色忠实度**：如果"所属 IP / 作品"不为"无"，说明该角色来自某个知名作品。此时角色的外观必须忠实于该作品中大众印象里的经典形象——包括标志性的发型、服装、配色、配饰等。例如"赛博朋克2077"中的强尼·银手必须有金属义肢手臂和摇滚风格，"漫威"中的灭霸必须是紫色皮肤和壮硕体型。即使角色名字本身就是知名角色（如容嬷嬷、灭霸、V），也必须按照大众熟知的经典形象设计，除非角色外观重点中有明确的不同指示。
