## 必须标记为青色的区域（除了上述不可行走区域之外的一切）

- **阴影下的地面** — 阴影只是光照效果，地面本身可走，必须标青色
- 室内地板
- 广场、庭院、开放区域

- 岩石旁的地面
- 所有其他未被明确标注为不可行走的区域

## 核心原则
**宁可多标，不可漏标。** 因为有的道路被树冠遮挡，或者道路从建筑物视觉高度的"背后"穿过，如果只标"能看到的路面"就会导致角色完全走不了。本 fallback 的目标是确保角色能走遍除了建筑物底部/水域/天空外的所有区域，不追求完美边界。

核心技巧：
- **树冠下** → 地面在下面，标青色
- **阴影下** → 标青色
- **建筑物向上延伸的部分** → 那是建筑背后的地面，标青色
- **建筑物只有底部约一人才是真的建筑**

## 标记颜色要求
- 使用**高饱和度的纯青色（Cyan）**进行叠加标记
- 具体色值：RGB(0, 255, 255)，即纯青色
- 不透明度：**70%-80%**（偏高，确保标记清晰可辨）
- 纯青色 RGB(0, 255, 255) 是一种**人工标记色**；请直接把它当作叠加层颜色使用，不要担心与原图风格是否一致
- 不要使用浅蓝、天蓝或蓝绿色，必须是鲜艳的纯青色 RGB(0, 255, 255)

## 其它要求
1. 生成一张与输入图片**完全一样**的地图，但在所有可行走区域上叠加上述青色标记
2. 建筑物只有底部约一人高度（16-24px）保留为建筑本身，其向上延伸的部分应标青色（那是建筑背后的地面/道路）
3. 水域、天空保持原样，不做任何修改
4. 建筑物入口/门口处应标青色（因为门是可进入的，属于可行走区域）
5. 桥面必须标记为青色
6. 特别注意树下的地面一定要标青色
